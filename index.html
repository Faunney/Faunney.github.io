【工具】tmux基本使用
Created2025-04-14|工具
使用场景例如：在服务器上跑一些耗时的程序或下载一些大文件，ssh断开的时候也希望持续执行任务。 此时可以用tmux开一个新的终端会话（session），只要服务器不关机且不关闭该会话，不管断开多长时间都可以再次访问到这个会话。 下载tmux12345sudo apt install tmux #Ubuntusudo yum install tmux #CentOSbrew install tmux #MacOS tmux基本命令1. 创建tmux新会话1tmux new -s [session_name] 2. 暂时退出当前tmux会话窗口1tmux detach 3. 查看当前所有tmux会话1tmux ls 4. 再次打开tmux会话1tmux a -t [session_name] 5. 关闭tmux会话1tmux kill-session -t [session_name]
